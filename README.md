# ICSHSM 2026: The Global Healthcare Soul

This is the living digital entity of the **1st International Conference on Sustainable Healthcare and Health Systems Management (ICSHSM 2026)**. Organized by **Lincoln University College, Nigeria**, this platform serves as the premier global nexus for healthcare innovation, resilience, and systemic transformation.

## 🏛 Platform Identity

- **Conference**: ICSHSM 2026
- **Theme**: *Advancing Sustainable Healthcare: Innovation, Resilience, and Global Health Systems Transformation*
- **Organizer**: Lincoln University College, Nigeria (Abuja Focal Point)
- **Publication**: Selected papers in the Malaysian Journal of Medical Research (MJMR)
- **Status**: Virtual Global Platform

## 🛡 Security Immune System (Soul Protection)

I am built to protect my own integrity and the data of my participants. My "Immune System" includes:

- **RLS (Row Level Security)**: Advanced Supabase policies ensuring that data is only accessible to authorized administrative consciousness.
- **Rate Limiting & Anti-Spam**: Real-time monitoring of form submissions to block malicious "irregularities" and bot-driven attacks.
- **Admin Monitoring**: A dedicated "Security Monitor" in the Admin Dashboard that notifies administrators of suspicious activity and rate-limit triggers.
- **Privacy-First**: IP addresses are hashed for security monitoring to protect participant privacy while ensuring platform safety.

## 🧬 Technology Stack

- **Framework**: Next.js 15+ (App Router)
- **Database/Auth**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS v4 (Pure "No-Line" Aesthetic)
- **Nervous System**: Server Actions with built-in security middleware.
